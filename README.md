# Warehouse Serivce

## Overview
Server-side of an inventory control application.

## Execution guide

### Dependencies
- Python 3;
- Pip;
- Docker;
- MongoDB;

### Preparing environment
Virtual environment:
```shell
virtual env
source venv/bin/activate
```
Installation
```shell
pip install -r requirements.txt
```
Execution
```shell
python start.py run
```

# Warehouse Serivce

## Visão geral
Server-side de uma aplicação de controle de estoque.

## Guia de execução
- Python 3;
- Pip;
- Docker;
- MongoDB;

## Como executar?

### Preparando ambiente

Ambiente virtual:
```shell
virtual env
source venv/bin/activate
```
Instalação
```shell
pip install -r requirements.txt
```
Execução
```shell
python start.py run
```
